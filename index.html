01 — Sobre mí

Backend, datos e infraestructura, en una sola persona.

Soy Ingeniero en Desarrollo de Software con experiencia practica en C#, PHP, Python y JavaScript, diseño y consulta de bases de datos relacionales en MySQL, y administración de infraestructura en Linux con Docker y Nginx. Programo aplicando programación orientada a objetos, integro APIs REST con manejo de JSON, y despliego sistemas completos de extremo a extremo: desde el código hasta el servidor donde corre.

También trabajo con análisis de datos y Business Intelligence — Looker Studio, Google Sheets avanzado, y en Python con Pandas y Scikit-learn para limpieza de datos, modelos predictivos y dashboards de KPIs. Uso Git y GitHub para versionar y documentar cada proyecto, y manejo SSH y PM2 para diagnóstico y mantenimiento de servidores en producción.
